# Learn Chinese

An interactive terminal game that teaches Mandarin Chinese from absolute beginner level. Built on [quest-engine](https://github.com/thorski1/quest-engine), guided by your friendly dragon companion **Long Long** (龙龙).

## What You'll Learn

| Pack | Topics |
|------|--------|
| **Pinyin** | The four tones, initials, finals, reading pinyin, tone sandhi rules |
| **Greetings** | Hello/goodbye, introductions, politeness, yes/no patterns |
| **Numbers** | 1-10, 11-99, 100+, measure words, money, time, dates |
| **Food & Drink** | Fruits, drinks, dishes, ordering food, tastes, cooking words |
| **Family** | Family members, relationships, talking about your family |
| **Daily Life** | Routines, home, school/work, hobbies, weather |
| **Travel** | Directions, transportation, hotels, sightseeing, emergencies |
| **Culture** | Festivals, traditions, history basics, arts, modern China |

Every challenge includes both **Chinese characters** and **pinyin** romanization so you can learn to read and pronounce from day one.

## Quick Start

```bash
# Install
pip install learn-chinese

# Play the full game
learn-chinese

# Play a single pack
pinyin-quest
greetings-quest
numbers-chinese-quest
food-drink-quest

# Launch the web version
learn-chinese --web
```

## Requirements

- Python 3.10+
- A terminal with UTF-8 support (for Chinese characters)

## How It Works

Each skill pack contains themed **zones** with quiz challenges. Answer questions correctly to earn XP, level up, and unlock new zones. Long Long provides hints when you're stuck and teaches you something new after every answer.

The game uses spaced repetition principles — concepts you struggle with appear more often, and each zone builds on what came before.
